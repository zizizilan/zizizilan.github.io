新建一篇博客

hexo n 或者 hexo new 后面加上名字就可以创建一篇博客
完整格式是：

布局

布局是定义生成的类型，有三种：post（发布）、page（页面） 和 draft(草稿)
在创建者三种不同类型的文件时，默认创建的是post类型（就是我们要发布的博客），它们将会被保存到不同的路径；而您自定义的其他布局和 post 相同，都将储存到 source/_posts 文件夹。
布局 路径
post source/_posts
page source
draft source/_drafts

文章标题

有空格的话，用引号吧文章标题括起来！！！

新建时候的其他参数

参数 描述
-p, –path 自定义新文章的路径
-r, –replace 如果存在同名文章，将其替换
-s, –slug 文章的 Slug，作为新文章的文件名和发布后的 URL

写作内容

新建一篇博客之后我们就可以对博客内容进行编辑，hexo支持markdown语法，假如不知道什么是markdown语法的话，也没关系，因为一般人的话会简单的使用的可以了，不用专门学习，尽管它不难，然后只要知道它对于程序员很优雅就好了

markdown基本语法

标题

在想要设置为标题的文字前面加#来表示，一个#是一级标题，两个#是二级标题，依此类推，支持六级标题
注意：标准语法一般在#后跟个空格再写文字，但是有的不用

字体

加粗：将要加粗的文字用两个 * 包起来
斜体：用一个 * 包起来
倾斜并且加粗：用三个 * 包起来
删除线：用两个 ～ 包起来
居中：用 <center></center> 包起来

引用

在引用的文字前面加>即可，引用也可以嵌套，

分割线

三个或者三个以上的-或者*都可以00000

图片

！[ 图片alt]（图片的地址”图片的title”）
图片alt；显示在图片下方的文字，相当于图片内容的解释
图片title是图片的标题，当鼠标指向图片时显示的内容，title可加可不加

超链接

[超链接名]（超链接地址"超链接title"）

列表

无序列表：用 - + * 任何一种都可以
- 列表内容 tips：-跟后面的内容有一个空格
有序列表：数字_点 列表内容;序号跟后面的内容之间有空格
1. 列表内容
列表嵌套：上一级和下一级敲三个空格即可

代码

单行代码：代码之间分别用一个反引号包起来(反引号在键盘的左上角，esc下面，记得切换为英文)
代码块：代码之间分别用三个反引号包起来，且两边的反引号单独占一行

hexo中间的空格

博客中间有空行，会只显示一行，就是说你编辑博客的时候连着回车两次，网页上只会显示一个空行，but 用<br>符号可以代表一个回车；
